# Personal Website Starter

This is a minimal personal website scaffold. Open `index.html` in a browser or use a development server.

Quick start:

```bash
# open locally (mac/windows with default apps)
start index.html
# or use a simple http server (Python)
python -m http.server 8000
```

Deployment:
- Push this repo to GitHub and enable GitHub Pages from the repository settings (use `main` branch and `/` root).

What I added:
- `index.html` — site structure and sections
- `styles.css` — responsive styles with light/dark variables
- `script.js` — theme toggle and smooth scroll
